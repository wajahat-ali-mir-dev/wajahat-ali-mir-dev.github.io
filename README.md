## 🚀 Getting Started

### Prerequisites

* Node.js 18+
* npm / yarn

---

### Installation

1️⃣ Clone Repository

```bash
git clone https://github.com/wajahat-ali-mir-dev/wajahat-ali-mir-dev.github.io.git
```

2️⃣ Enter Folder

```bash
cd wajahat-ali-mir-dev.github.io
```

3️⃣ Install Packages

```bash
npm install
```

4️⃣ Run Locally

```bash
npm run dev
```

---

## 📦 Build & Deploy

Build production files:

```bash
npm run build
```

Deploy to GitHub Pages:

```bash
npm run deploy
```
